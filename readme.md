# `rbtv-mediathek.pyz download --help`
```
usage: rbtv-mediathek.pyz download [-h]
                                   (--episode EPISODE | --season SEASON | --show SHOW)
                                   [--format FORMAT] [--outdirtpl OUTDIRTPL]

optional arguments:
  -h, --help            show this help message and exit
  --episode EPISODE     Download this episode
  --season SEASON       Download all episodes of this season
  --show SHOW           Download all episodes of this show
  --format FORMAT       youtube-dl format string
  --outdirtpl OUTDIRTPL
```
